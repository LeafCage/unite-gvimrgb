# unite-gvimrgb
GVimが標準で用意している、`:highlight` コマンドで利用可能な色名 [1] を、  
色見本付きで unite.vim で一覧する unite-source です。  
動作には `unite.vim` が必要です。  
  
[1] $VIMRUNTIME/rgb.txt に載っている色名一覧

## Usage
- `:Unite gvimrgb`

- `:Unite gvimrgb:sort`
